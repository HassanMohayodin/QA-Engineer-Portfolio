# Demo Web Shop Manual Testing Portfolio

## Project Overview

This repository contains a complete manual testing project performed on the Demo Web Shop e-commerce application.

The purpose of this project was to simulate a real-world Software Quality Assurance (SQA) process by creating and executing test cases, reporting defects, preparing test documentation, and maintaining requirement traceability.

---

## Application Under Test

**Application:** Demo Web Shop
**Type:** E-Commerce Web Application
**Website:** https://demowebshop.tricentis.com/

---

## Project Objectives

The objectives of this project were:

* Validate core business functionalities.
* Identify defects and usability issues.
* Practice professional QA documentation.
* Demonstrate manual testing skills using a real application.

---

## Scope of Testing

The following modules were tested:

* User Registration
* User Login
* Product Search
* Shopping Cart
* Wishlist
* Checkout Process
* Logout Functionality
* Basic UI Validation
